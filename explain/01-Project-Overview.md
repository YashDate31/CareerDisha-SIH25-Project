## What is CareerDisha?

**CareerDisha** is a comprehensive career guidance platform designed specifically for Indian students and their parents. The name "CareerDisha" combines "Career" (profession/job) and "Disha" (direction in Hindi), meaning "Career Direction" - helping students find the right path for their future.

## ğŸ’» **Technical Overview** (Simplified)

**What is it built with?**
- **Frontend:** React.js (JavaScript framework for user interface)
- **Backend:** Django (Python web framework for server-side logic)
- **Database:** SQLite/PostgreSQL (stores all data - users, content, etc.)

1. Students visit the website on their phone/computer
2. They can browse resources, take quizzes, search colleges
3. All data is fetched from a backend server
4. Administrators can add new content through an admin panel
5. Everything works together to provide a seamless experience
